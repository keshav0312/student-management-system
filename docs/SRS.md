# SRS - Student Management System

## 1. Introduction
This is a simple console-based project made in Java.
The system is used to store and manage student details like name, email, and course.

## 2. Objective
The main objective of this project is to learn Java basics, CRUD operations, and later learn how to connect Java with MySQL using JDBC.

## 3. Features
1. Add Student
2. View All Students
3. Search Student by Name Or Id
4. Update Student Details
5. Delete Student

## 4. Tools and Technologies
- Java
- MySQL Database
- JDBC (Java Database Connectivity)
- Maven (Project Build Tool)

## 5. Student Data Fields
- ID (Auto Increment)
- First Name
- Last Name
- Email
- Age
- course
- gender
- address
- mobile Number

## 6. Working
Data is stored permanently in MySQL Database using JDBC.

## 7. Conclusion
This project helps understand how to build a small application using Java and 
understand oops concepts how it works in and what is role in the real world projects ,we also focus on how to handle exception also how to store data using a database ....
