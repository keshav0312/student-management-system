Project: Console Student Management System

Purpose: Maintain student records via console

Actors: Admin (user of console app)

Features (Functional):

Add Student (id, firstName, lastName, email, age)

View Student by id

Update Student

Delete Student by id

List all students

Non-functional:

Use JDBC + MySQL

Use PreparedStatement for SQL queries

Console-based (no GUI)

Deliverables:

Source code (Maven), schema.sql, README, short demo screenshot
